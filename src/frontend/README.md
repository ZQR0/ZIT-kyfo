# KYFO — руководство пользователя

## О чём этот сайт

**KYFO** — веб-система для авиакомпаний и кафе в аэропортах.

Если рейс задерживается, авиакомпания начисляет пассажирам деньги на **посадочный талон**. Эти деньги можно потратить в кафе или другой точке обслуживания в аэропорту — как электронный кошелёк, привязанный к номеру талона.

На сайте два раздела:

| Раздел | Кто пользуется | Зачем |
|--------|----------------|-------|
| **Личный кабинет авиакомпании** | сотрудник авиакомпании | смотреть рейсы и талоны, начислять компенсацию при задержке |
| **Терминал кафе** | кассир в кафе | проверить баланс талона и принять оплату |

---

## Как открыть сайт

1. Убедитесь, что запущены база данных и сервер приложения (инструкция — в корне проекта).
2. Откройте браузер и перейдите по адресу:

   **http://localhost:8080/**

## Общая навигация

Вверху страницы две вкладки:

- **Личный кабинет авиакомпании** — нужен вход по логину и паролю.
- **Терминал кафе** — работает сразу, без входа.

Переключайтесь между вкладками, нажимая на их названия.

В правом нижнем углу иногда появляются **всплывающие сообщения** (например, «Добро пожаловать» или текст ошибки). Они исчезают через несколько секунд.

## Личный кабинет авиакомпании

### Вход в систему

1. Откройте вкладку **«Личный кабинет авиакомпании»**.
2. Введите **логин** и **пароль**.
3. Нажмите **«Войти»**.

При успешном входе форма входа скроется, появится рабочая область с названием вашей авиакомпании.

#### Тестовые учётные записи

Для проверки можно использовать любой из логинов ниже. Пароль у всех одинаковый: **`password`**

| Логин | Авиакомпания |
|-------|--------------|
| `aeroflot` | Аэрофлот |
| `pobeda` | Победа |
| `s7` | S7 Airlines |
| `ural` | Уральские авиалинии |
| `rossiya` | Россия |
| `nordwind` | Nordwind Airlines |
| `azimuth` | Азимут |
| `redwings` | Red Wings |
| `utair` | UTair |
| `smartavia` | Smartavia |

Если логин или пароль неверные, внизу экрана появится сообщение об ошибке.

### Выход

Нажмите кнопку **«Выйти»** справа вверху рабочей области. После этого снова отобразится форма входа.

---

### Таблица «Рейсы»

После входа автоматически загружается список всех рейсов в системе.

| Колонка | Что означает |
|---------|--------------|
| **ID** | номер рейса в системе (нужен для начисления компенсации) |
| **Авиакомпания** | кто выполняет рейс |
| **Маршрут** | откуда → куда (коды аэропортов, например SVO → LED) |
| **Вылет** | плановое время вылета |
| **Прилёт** | плановое время прилёта |
| **Задержка** | сколько минут рейс задержан; если задержки нет — стоит «—». Задержанные рейсы выделены красным |

Кнопка **«Обновить»** перезагружает таблицу, если данные могли измениться.

---

### Таблица «Посадочные талоны»

Здесь отображаются все посадочные талоны пассажиров.

| Колонка | Что означает |
|---------|--------------|
| **Номер** | уникальный номер талона (им пользуется кафе) |
| **Рейс** | ID рейса, на который оформлен талон |
| **Место** | место в самолёте (например, 12A) |
| **Баланс** | сколько денег сейчас на талоне |

Кнопка **«Обновить»** обновляет список. После начисления компенсации балансы в таблице обновятся автоматически.

#### Примеры номеров талонов для проверки

| Номер талона | Рейс | Место |
|--------------|------|-------|
| `TKT202607140001` | 1 | 12A |
| `TKT202607140004` | 3 | 8D |
| `TKT202607140007` | 5 | 3A |

---

### Начисление компенсации

Эта форма нужна, когда рейс задержан и авиакомпания хочет **начислить деньги всем пассажирам этого рейса**.

Заполните три поля:

| Поле | Что вводить |
|------|-------------|
| **Рейс** | выберите из списка: авиакомания, маршрут, дата и время (задержанные — в начале списка, с пометкой «+Nм») |
| **Сумма** | сколько рублей начислить **каждому** пассажиру рейса (например, `500`) |
| **Точка обслуживания** | выберите из списка (название и ID в скобках, например: «Кафе «Небо» (1)») |

Нажмите **«Начислить»**.

После успешного начисления:
- появится сообщение с результатом;
- таблица талонов обновится — балансы пассажиров этого рейса увеличатся.

#### Пример

1. В таблице «Рейсы» найдите рейс с ID **3** (есть задержка 45 мин).
2. В форме выберите рейс с задержкой (например, **Победа · SVO→KZN · 14.07, 12:00 · +45м**), сумма **500**, точка **Кафе «Небо» (1)**.
3. Нажмите **«Начислить»**.
4. Откройте таблицу «Посадочные талоны» — у талонов рейса 3 баланс вырос.

#### ID точек обслуживания (для поля «ID точки»)

| ID | Название |
|----|----------|
| 1 | Кафе «Небо» |
| 2 | Кофейня «Высота» |
| 3 | Бистро «Полёт» |
| 5 | Кафе «Север» |
| 9 | Кафе «Черноморье» |

Полный список точек — в базе данных (ID от 1 до 20).

## Терминал кафе

Раздел для кассира в аэропорту. **Вход не требуется** — переключитесь на вкладку **«Терминал кафе»**.

Пассажир называет **номер посадочного талона** (можно показать на телефоне или назвать вслух). Кассир вводит его в форму.

---

### Проверка баланса

1. В поле **«Номер талона»** введите номер, например: `TKT202607140001`
2. Нажмите **«Проверить»**
3. Под формой появится строка вида: **«Баланс: 500,00 ₽»**

Если талон не найден, сообщение будет красным с текстом ошибки.

---

### Оплата

Списание денег с талона пассажира за покупку в кафе.

1. **Номер талона** — номер посадочного талона пассажира
2. **Сумма** — стоимость покупки (например, `150`)
3. **Точка обслуживания** — выберите своё кафе из списка (название и ID в скобках, например: «Кафе «Небо» (1)»)

Нажмите **«Списать»**.

При успешной оплате под формой появится сообщение с **новым балансом** талона.

Если на талоне недостаточно денег, появится сообщение об ошибке — оплата не пройдёт.

В списке **«Точка обслуживания»** отображаются только активные кафе. Формат каждого пункта: **название (ID)**, например `Кофейня «Высота» (2)`.

#### Пример сценария «от начисления до покупки»

**Шаг 1 — авиакомпания начисляет компенсацию**

1. Войдите как `aeroflot` / `password`.
2. Начислите 500 ₽ на рейс **3** (ID точки: **1**).

**Шаг 2 — пассажир покупает в кафе**

1. Перейдите на вкладку **«Терминал кафе»**.
2. Проверьте баланс талона `TKT202607140004` — должны увидеть обновлённую сумму.
3. Списание: талон `TKT202607140004`, сумма `150`, точка **Кафе «Небо» (1)**.
4. Проверьте баланс снова — он уменьшится на 150 ₽.

## Полный сценарий «с нуля» за 2 минуты

1. Откройте http://localhost:8080/
2. Войдите: логин `aeroflot`, пароль `password`.
3. Посмотрите таблицу **«Рейсы»** — найдите рейсы с красной задержкой.
4. Посмотрите таблицу **«Посадочные талоны»** — запомните любой номер талона.
5. Начислите **300 ₽** на рейс **3** (ID точки **1**).
6. Перейдите в **«Терминал кафе»**.
7. Проверьте баланс талона `TKT202607140004`.
8. Списание **100 ₽** с того же талона (ID точки **1**).
9. Снова проверьте баланс — сумма уменьшилась.
10. Вернитесь в кабинет и нажмите **«Обновить»** в таблице талонов — баланс совпадёт.
